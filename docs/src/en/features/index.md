In this section, we'll cover (almost) all of Stylo's features. If you're wondering what Stylo can do and how, you'll find the answer here (on the pages shown on the bottom left on desktop, and on the menu on smartphone). If you see one or more features not listed here, please let us know so we can add them! If you need help getting to grips with Stylo's features, you can visit the community support forum at [this link](https://discussions.revue30.org/c/stylo/entraide-stylo/7) and chat there with other users.
