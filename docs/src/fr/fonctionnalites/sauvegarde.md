---
title: "Sauvegarder ses données"
---

Stylo ne constitue pas un service d'hébergement de données. Il est important de faire des sauvegardes régulières des documents sur Stylo, notamment pour les données sensibles ou importtantes pour un collectif de recherche. 

Pour le moment, vous pouvez sauvegarder vos données en regroupant les articles importants/sensibles d'un espace de travail personnel ou partagé dans un corpus, puis en exportant les fichiers sources du corpus (Markdown pour le texte, YAML pour les métadonnées, et BibTeX pour les références bibliographiques). 

Prochainement, il sera possible d'exporter l'ensemble de ses données sur Stylo en lot et de manière facilitée pour les utilisateurs·ices.
